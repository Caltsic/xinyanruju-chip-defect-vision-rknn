# WS2812 环形补光 SPI 部署记录

更新时间：2026-05-03

## 接线

WS2812-8 环形灯只需要接输入端 `DI`，`DO` 不需要接开发板。

```text
WS2812 VCC / 5V  -> 泰山派 40Pin 物理 2 或 4 脚 5V
WS2812 GND       -> 泰山派 40Pin 物理 20 脚 GND
WS2812 DI / DIN  -> 泰山派 40Pin 物理 19 脚 SPI1_MOSI(M1) / GPIO2_C2
WS2812 DO / DOUT -> 不接；只在串接下一组灯时接下一组 DI
```

建议：

- `DI` 串联 330-470 欧电阻。
- 灯环 5V 和 GND 之间并 470uF 左右电容。
- 泰山派 IO 是 3.3V，WS2812 供电是 5V。少量灯珠常可识别 3.3V 数据，但稳妥做法是加 74HCT/74AHCT 电平转换。
- 8 颗 WS2812 全白满亮约 480mA，不建议长时间满亮直接从板子 5V 排针取电；高亮时使用外部 5V，并与泰山派共地。

## 使用命令

低亮度白光：

```powershell
F:\anaconda\python.exe .\tools\adb_ws2812_ring.py set --rgb 255,255,255 --brightness 0.08 --count 8
```

关灯：

```powershell
F:\anaconda\python.exe .\tools\adb_ws2812_ring.py off
```

部署或重装：

```powershell
F:\anaconda\python.exe .\tools\adb_ws2812_ring.py deploy --reboot
```

状态检查：

```powershell
F:\anaconda\python.exe .\tools\adb_ws2812_ring.py status
```
